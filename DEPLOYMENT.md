# Frenaphok Website - Deployment Guide

## Quick Start

Your Frenaphok website is fully built and ready to deploy. Follow these steps to get it live.

## Prerequisites

- GitHub account
- Vercel account (free: vercel.com)

## Deployment Steps

### Option 1: Deploy on Vercel (Recommended)

1. **Push to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Frenaphok website"
   git branch -M main
   git remote add origin https://github.com/your-username/frenaphok-website.git
   git push -u origin main
   ```

2. **Connect to Vercel**
   - Go to https://vercel.com/new
   - Import the GitHub repository
   - Click "Deploy"
   - Your site will be live in seconds!

### Option 2: Deploy Manually

1. Build the project:
   ```bash
   pnpm build
   ```

2. The build output in `.next` is ready to deploy to any Node.js hosting

## Custom Domain

1. In Vercel dashboard, go to Project Settings → Domains
2. Add your custom domain (e.g., frenaphok.com)
3. Update your domain's DNS settings to point to Vercel
